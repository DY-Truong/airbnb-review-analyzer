# Airbnb Review Analyzer (NLP)

This project analyzes Airbnb guest reviews to identify patterns in negative experiences using natural language processing techniques.

## Overview
The goal of this project is to extract insights from Airbnb reviews by performing sentiment analysis and identifying common complaint themes.

## Dataset
- Airbnb Reviews dataset
- 300K+ reviews
- Focused on textual comments

## Data Cleaning
- Removed missing and short reviews
- Converted text to lowercase
- Removed punctuation and stopwords
- Filtered non-English text

## Methods
- Sentiment Analysis (Syuzhet / sentimentr)
- TF-IDF for keyword importance
- LDA Topic Modeling for discovering themes

## Key Findings
Negative reviews most commonly involved:
- Cleanliness issues
- Poor communication with hosts
- Check-in problems
- Property condition mismatches

## Tech Stack
- R
- NLP libraries
- TF-IDF
- LDA Topic Modeling
